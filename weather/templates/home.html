MetOffice Project

How to:
  1. Navigate to Data Operations link at the Navigation bar.

  2. Select 'All Countries' and 'All Readings' from drop-down list, and click Load Database.
    3. Note:This will take some time, as it will download data from internet and store it into database. Sit back and enjoy your cup of Coffee, while the MetOffice project App will do the heavy lifting for you.

  3. Navigate to Present tab, where you can visualize the weather readings of different readings and compare them. They are intuitive, so feel free to try experimenting with different values, and see how the output chart changes!

  4. Met Office will update data of a month only when it is over, so if you want to import newly added data into your database, use Update Database in the Data Operations tab
    5. Note: This will only look for the latest data. Duplicates entries will not be created, and it's less time consuming than Load Database option, which can also perform the same operation.
  5. Check About Page
